# Cryptocurrency-track

Удобное отслеживание актуальной криптовалюты

## Возможности

1. Удобная таблица отслеживания топ-100 лучших монет
2. Поиск
3. Отслеживаемое

## Запуск проекта

1. Установить зависимости - `npm install`
2. Запустить проект - `npm run dev`
